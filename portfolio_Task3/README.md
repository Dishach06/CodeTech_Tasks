# Disha's Portfolio

A modern, responsive personal portfolio website showcasing frontend development skills and projects.

## 🚀 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Dark Mode**: Toggle between light and dark themes
- **Smooth Animations**: Scroll reveal animations using ScrollReveal.js
- **Interactive Navigation**: Sticky navbar with active section highlighting
- **Mobile Menu**: Hamburger menu for mobile devices
- **Social Links**: Direct links to LinkedIn, Twitter, GitHub, and email
- **CV Download**: Direct link to resume

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties, flexbox, grid, and animations
- **JavaScript**: DOM manipulation and event handling
- **Boxicons**: Icon library for UI elements
- **Font Awesome**: Additional icon set
- **Google Fonts**: Custom typography (Poppins and Spartan)
- **ScrollReveal.js**: Animation library for scroll effects

## 📱 Sections

1. **Home**: Introduction and profession showcase
2. **About**: Personal background and education
3. **Services**: Offered development services
4. **Projects**: Portfolio of completed work
5. **Contact**: Contact information and form

## 🏃‍♀️ How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd portfolio_Task3
   ```

2. Open `index.html` in your web browser

No build process or dependencies required - it's a pure static website!

## 🎨 Customization

- **Colors**: Modify CSS custom properties in `:root` selector
- **Content**: Update text and links in `index.html`
- **Styling**: Edit `index.css` for layout and appearance changes
- **Functionality**: Modify `index.js` for behavior changes
